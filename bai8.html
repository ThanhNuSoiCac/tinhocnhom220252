Định Dạng Văn Bản với CSS

1. Định Dạng Màu Sắc Văn Bản

Với CSS, bạn có thể thay đổi màu sắc của văn bản bằng thuộc tính color. Ví dụ dưới đây thay đổi màu của văn bản thành màu đỏ:

Ví dụ: Đổi Màu Sắc Văn Bản

p {
color: red;
}

Đoạn văn bản này có màu chữ là đỏ.

2. Định Dạng Font Chữ

CSS cho phép bạn thay đổi font chữ của văn bản bằng thuộc tính font-family. Ví dụ dưới đây thay đổi font chữ của văn bản thành Arial:

Ví dụ: Đổi Font Chữ

p {
font-family: Arial, sans-serif;
}

Đoạn văn bản này sử dụng font chữ Arial.

3. Định Dạng Kích Thước Chữ

Có thể thay đổi kích thước chữ bằng thuộc tính font-size. Dưới đây là ví dụ thay đổi kích thước chữ:

Ví dụ: Đổi Kích Thước Chữ

p {
font-size: 24px;
}

Đoạn văn bản này có kích thước chữ là 24px.

4. Căn Chỉnh Văn Bản

Có thể căn chỉnh văn bản theo nhiều kiểu khác nhau: căn trái, căn giữa, hoặc căn phải bằng thuộc tính text-align.

Ví dụ: Căn Chỉnh Văn Bản

p {
text-align: center;
}

Đoạn văn bản này được căn giữa.

5. Định Dạng Đậm, Nghiêng và Gạch Chân

CSS cũng cho phép bạn làm cho văn bản trở nên đậm, nghiêng hoặc gạch chân. Dưới đây là ví dụ áp dụng các thuộc tính này:

Ví dụ: Định Dạng Đậm, Nghiêng và Gạch Chân

p {
font-weight: bold;
font-style: italic;
text-decoration: underline;
}

Đoạn văn bản này có kiểu chữ đậm, nghiêng và có gạch chân.

6. Khoảng Cách Giữa Các Dòng

Với thuộc tính line-height, bạn có thể điều chỉnh khoảng cách giữa các dòng chữ. Điều này rất hữu ích khi bạn muốn văn bản dễ đọc hơn.

Ví dụ: Khoảng Cách Dòng

p {
line-height: 1.8;
}

Đoạn văn bản này có khoảng cách giữa các dòng chữ là 1.8.
