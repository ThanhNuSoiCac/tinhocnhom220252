Định Dạng Văn Bản bằng CSS

1. Định Dạng Màu Sắc và Font Chữ

Với CSS, bạn có thể thay đổi màu sắc và font chữ của văn bản để làm nổi bật hoặc thay đổi phong cách của trang web. Ví dụ dưới đây sẽ minh họa cách thay đổi màu sắc và font chữ:

Ví dụ 1: Đổi Màu Sắc và Font Chữ


p {
    color: #FF5733; /* Màu chữ cam */
    font-family: 'Arial', sans-serif; /* Font chữ Arial */
}

Đoạn văn bản này có màu sắc là cam và font chữ là Arial.

2. Định Dạng Kích Thước Chữ

CSS cho phép bạn thay đổi kích thước chữ để nội dung dễ đọc hơn và nổi bật hơn. Dưới đây là cách thay đổi kích thước chữ:

Ví dụ 2: Đổi Kích Thước Chữ


p {
    font-size: 24px; /* Kích thước chữ 24px */
}

Đoạn văn bản này có kích thước chữ là 24px.

3. Định Dạng Căn Chỉnh Văn Bản

Có thể căn chỉnh văn bản theo nhiều hướng khác nhau: căn trái, căn giữa hoặc căn phải.

Ví dụ 3: Căn Chỉnh Văn Bản


p {
    text-align: center; /* Căn giữa văn bản */
}

Đoạn văn bản này được căn giữa.

4. Định Dạng Khoảng Cách và Line-Height

CSS cũng cho phép bạn điều chỉnh khoảng cách giữa các dòng chữ (line-height) và giữa các đoạn văn bản (margin, padding).

Ví dụ 4: Khoảng Cách Dòng và Đoạn Văn


p {
    line-height: 1.8; /* Khoảng cách dòng */
    margin-bottom: 20px; /* Khoảng cách giữa các đoạn văn */
}

Đoạn văn bản này có khoảng cách dòng lớn hơn và có khoảng cách giữa các đoạn văn bản.

5. Định Dạng Đậm, Nghiêng và Gạch Chân

CSS cho phép bạn làm cho văn bản trở nên đậm, nghiêng hoặc gạch chân để làm nổi bật các phần nội dung quan trọng.

Ví dụ 5: Định Dạng Đậm, Nghiêng và Gạch Chân


p {
    font-weight: bold; /* Làm đậm văn bản */
    font-style: italic; /* Làm nghiêng văn bản */
    text-decoration: underline; /* Gạch chân văn bản */
}

Đoạn văn bản này là đậm, nghiêng và có gạch chân.

6. Định Dạng Các Đoạn Danh Sách

CSS cũng cho phép bạn thay đổi cách hiển thị các danh sách (unordered list hoặc ordered list) trên trang web.

Ví dụ 6: Định Dạng Danh Sách


ul {
    list-style-type: square; /* Đổi kiểu điểm đầu của danh sách thành hình vuông */
    margin-left: 20px; /* Khoảng cách bên trái danh sách */
}
