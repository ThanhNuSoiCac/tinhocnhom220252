Luyện tập 1 trang 105 – Tin học 12

Tạo trang dang_ki.html chứa biểu mẫu đăng kí câu lạc bộ và bổ sung liên kết tới trang dang_ki trong phần cuối trang của tất cả các trang.

Lời giải

  • Tạo tệp HTML mới và đặt tên là dang_ki.html.
  • Trong tệp này, tạo biểu mẫu đăng kí câu lạc bộ bằng các thẻ <form>, <input>, <button>.
  • Thêm CSS để định dạng giao diện biểu mẫu cho đẹp và dễ sử dụng.
  • Ở phần cuối trang của tất cả các trang trong website, thêm liên kết dẫn tới trang dang_ki.html.

Luyện tập 2 trang 105 – Tin học 12

Thay đổi định dạng và màu sắc của phông chữ khi di chuyển chuột qua.

Lời giải

Có thể sử dụng hiệu ứng :hover trong CSS.

.vung {
  font-family: Arial, sans-serif;
  font-size: 14px;
  color: #333;
}

.vung:hover {
  font-size: 16px;
  color: red;
  font-weight: bold;
}

Vận dụng trang 105 – Tin học 12

Đưa ra một thiết kế khác cho website đã tạo ở phần Thực hành và chuyển toàn bộ website sang định dạng mới.

Lời giải

  • Tạo một tệp CSS mới, ví dụ: style.css.
  • Viết các quy tắc CSS theo thiết kế mới.
  • Liên kết tệp CSS mới vào tất cả các trang của website.

Ví dụ định dạng CSS:

/* Banner */
.banner {
  background: url("../assets/img/bg-masthead.jpg") no-repeat center;
  background-size: cover;
  padding: 12rem 0;
  text-align: center;
  color: darkred;
}

.banner h2 {
  font-size: 36px;
  font-weight: bold;
}

/* Slogan */
.slogan {
  background-color: #f8f9fa;
  text-align: center;
  padding: 7rem 3rem;
}

.row {
  display: flex;
  flex-wrap: wrap;
}

.block_3 {
  flex: 0 0 33.33%;
}

.block_3 h3 {
  font-size: 24px;
  font-weight: bold;
}

.text-content {
  font-size: 16px;
  color: #333;
}
